# 🖼️ PhotoEditor

**PhotoEditor** — это веб-приложение для редактирования и слияния фотографий с поддержкой множества фильтров и эффектов. Работает полностью в браузере без необходимости установки и без серверной части.

---

## ✨ Основные возможности

### 🖼️ Работа со слоями

- **Загрузка изображений**: Поддержка 2 слоёв с возможностью загрузки через клик или drag-and-drop
- **Управление слоями**: Выбор активного слоя, удаление слоёв
- **Позиционирование**: Перемещение изображений на холсте с помощью мыши
- **Трансформация**:
  - Масштабирование (scale)
  - Поворот (rotation)
  - Отражение по горизонтали (flipX)
  - Регулировка прозрачности (opacity)

### 🎨 Фильтры

| Фильтр | Описание |
|--------|----------|
| **Яркость** (Brightness) | Регулировка яркости изображения |
| **Контраст** (Contrast) | Увеличение или уменьшение контраста |
| **Насыщенность** (Saturation) | Управление интенсивностью цветов |
| **Температура** (Temperature) | Тёплые и холодные тона |
| **Оттенок** (Hue) | Изменение цветового тона |

### ✨ Эффекты

| Эффект | Описание |
|--------|----------|
| **Размытие** (Blur) | Мягкое размытие изображения |
| **Резкость** (Sharpness) | Увеличение чёткости деталей |
| **Виньетка** (Vignette) | Затемнение краёв изображения |
| **HDR** | Эффект расширенного динамического диапазона |
| **Зернистость** (Grain) | Добавление плёночной зернистости |

### 🌀 Искажения

| Эффект | Описание |
|--------|----------|
| **Закручивание** (Twist) | Скручивание от центра |
| **Выпуклость** (Bulge) | Эффект "рыбий глаз" |
| **Вогнутость** (Pinch) | Сжатие к центру |
| **Волна** (Wave) | Синусоидальная волна (горизонтальная / вертикальная) |
| **Рябь от центра** (Radial Ripple) | Концентрические круги от центра |
| **Кривое зеркало** (Funhouse) | Комбинированное искажение |
| **Водоворот** (Swirl) | Радиальное закручивание |

### 🖌️ Инструмент ластика

| Функция | Описание |
|---------|----------|
| **Кисть** | Настраиваемый размер (1-200px) |
| **Режимы** | Стирание / Восстановление / Умный ластик |
| **Мягкость** | Жёсткая / Мягкая кисть (0-100%) |
| **Прозрачность** | Регулировка силы стирания (10-100%) |
| **Умный ластик** | Удаление похожих цветов в зоне клика |
| **История** | Отмена/Повтор до 20 действий |
| **Горячие клавиши** | Ctrl+Z, [/], E, ПКМ |

### 🎮 Пикселизация

| Режим | Описание |
|-------|----------|
| **Мозаика** | Простая пикселизация блоками |
| **Pixel Art** | С дизерингом Floyd-Steinberg |
| **Ретро-палитра** | Game Boy / NES / C64 стиль |

### 🎭 Пресеты фильтров

| Пресет | Описание |
|--------|----------|
| **Нет** (None) | Сброс всех фильтров |
| **Чёрно-белый** (B&W) | Обесцвечивание с усилением контраста |
| **Сепия** (Sepia) | Винтажный эффект в тёплых тонах |
| **Тёплый** (Warm) | Тёплые оттенки с увеличением яркости |
| **Холодный** (Cold) | Холодные тона с контрастом |
| **Винтаж** (Vintage) | Приглушённые цвета с тёплым оттенком |

### 🌟 Пресеты эффектов

| Пресет | Описание |
|--------|----------|
| **Нет** (None) | Сброс всех эффектов |
| **Мягкий** (Soft) | Лёгкое размытие с виньеткой |
| **Драматичный** (Dramatic) | Высокий контраст с HDR и виньеткой |
| **Мечтательный** (Dreamy) | Размытие с увеличением яркости |
| **Грубый** (Gritty) | Зернистость с резкостью и высоким контрастом |
| **Кино** (Cinema) | Кинематографический эффект с виньеткой |

### 🔀 Режимы наложения (Blend Modes)

- Обычный (Normal / Source-over)
- Умножение (Multiply)
- Экран (Screen)
- Наложение (Overlay)
- Темнее (Darken)
- Светлее (Lighten)
- Dodge
- Жёсткий свет (Hard Light)

### 🔧 Интерфейс

- **Масштабирование UI**: Изменение размера интерфейса (50–200%)
- **Масштабирование холста**: Zoom холста (25–300%)
- **Центрирование**: Быстрое центрирование холста
- **Табы**: Переключение между базовыми настройками, фильтрами и эффектами
- **Горизонтальный скролл вкладок**: Плавная прокрутка панели вкладок на мобильных устройствах
- **Изменяемый сайдбар**: Регулировка ширины боковой панели
- **Экспорт**: Сохранение результата в формате PNG
- **Focus Mode**: Режим без отвлечений с плавающей панелью слоёв и мини-тулбаром (клавиша `F`)

## 🎯 Особенности

- Работает полностью в браузере (без серверной части)
- Поддержка мобильных устройств с адаптивным интерфейсом
- Поддержка жестов (pinch-to-zoom)
- Drag-and-drop загрузка файлов
- Реактивное обновление превью
- Улучшенное размытие краёв при удалении фона с нелинейной интерполяцией и гауссовым сглаживанием

## 💾 Технологии

- **HTML5** — разметка и Canvas API
- **CSS3** — стилизация и адаптивный интерфейс
- **JavaScript** (без фреймворков) — вся логика приложения
- **Canvas API** — рендеринг и применение фильтров

## 🚀 Установка и использование

Приложение не требует установки и сборки — достаточно открыть `index.html` в браузере.

```bash
# Клонируйте репозиторий
git clone https://github.com/berlandbors/PhotoEditor.git

# Откройте приложение в браузере
open index.html
```

Или просто скачайте ZIP-архив репозитория и откройте файл `index.html`.
